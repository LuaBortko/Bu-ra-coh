### Introdução
- Qual a ideia principal do seu jogo? 
    Ser um jogo de estratégia em turnos, pvp, baseado no clássico jogo de cartas Buraco
- Que tipo de experiência você quer dar para o seu jogador?
    Queremos que o jogador sinta que cada carta importa. Que cada turno seja uma decisão estratégica entre risco e recompensa, combinando lógica de cartas com combate PvP dinâmico. O jogador deve sentir pressão constante para decidir entre guardar cartas para combos maiores ou jogar para sobreviver. Além disso, personagens com habilidades únicas permitem diferentes estilos de jogo, incentivando adaptação e planejamento a cada partida.
